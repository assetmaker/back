# Asset_Maker backend

이 프로젝트는 생성형 인공지능을 활용하여 게임 및 인터랙티브 콘텐츠 제작에 필요한 다양한 에셋을 자동 생성하는 시스템입니다. 

백엔드에서는 ComfyUI, Meshy, OpenAI GPT API를 통합하여 2D 픽셀 에셋 생성, 이미지 변환(img2img), 3D 모델 생성, 스크립트 생성 기능을 제공합니다.

두 리포지토리는 함께 동작하며 다음과 같은 목적을 갖습니다:

- 텍스트 또는 이미지 입력만으로 고품질 2D/3D 에셋을 자동 생성
- ComfyUI SaveImageWebsocket 프로토콜 기반 실시간 이미지 수신
- 프론트엔드에서 직접 생성 진행도 표시 및 결과 다운로드 제공
- 3D 모델 및 AI 스크립트 생성 기능 포함한 확장형 제작 도구

전체적으로, Asset_Maker 프로젝트는 개발자·디자이너·크리에이터가 보다 빠르고 효율적으로 에셋을 제작할 수 있도록 설계된 **AI 기반 멀티 에셋 생성 플랫폼**입니다.
